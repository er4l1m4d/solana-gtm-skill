# Social Content Planner — 30-Day Content Calendar

Generate a structured 30-day content calendar for Solana project launches. Includes Solana-specific hooks, platform strategies, and engagement tactics.

---

## When to Use

Use this when a user says:
- "Create a content calendar"
- "What should I post for my launch?"
- "Social media plan for my project"
- "Help me plan my launch content"
- "Content strategy for X/Twitter"

## Execution Mode

First, confirm the user's mode:

**Hackathon Mode (2-3 weeks)**
- Focus: Quick community setup, demo day hype
- Timeline: Compress to 2-week calendar
- Skip: Long-form content, complex campaigns

**Founder Mode (3-6 months)**
- Focus: Sustainable content rhythm, brand building
- Timeline: Full 30-day calendar + monthly themes
- Include: Multi-platform strategy

## Platform-Specific Guidelines

### X/Twitter
**Best Practices:**
- Thread length: 5-10 tweets (sweet spot for engagement)
- Tweet timing: 9-11 AM EST, 1-3 PM EST (peak crypto hours)
- Use visuals: Charts, screenshots, demo videos
- Engagement: Reply to every comment in first hour
- Hashtags: 2-3 per tweet max

**Thread Structure:**
```
Tweet 1: Hook (problem statement or bold claim)
Tweet 2-3: Context (why this matters)
Tweet 4-6: Solution (what you're building)
Tweet 7-8: How it works (technical details)
Tweet 9: Traction / social proof
Tweet 10: CTA (link, join, follow)
```

### Discord
**Best Practices:**
- Channels: #general, #announcements, #support, #feedback, #dev-updates
- Roles: Member, Early Adopter, Beta Tester, Contributor
- Events: Weekly AMAs, monthly community calls
- Moderation: Clear rules, active moderation, responsive team

### Telegram
**Best Practices:**
- Use for announcements only (not discussion)
- Pin important messages
- Link to Discord for discussion
- Keep it clean and focused

### YouTube
**Best Practices:**
- Video length: 3-10 minutes (tutorial depth)
- Thumbnail: Bold text, clean design
- Description: Links to product, social, documentation
- End screen: Subscribe + next video

## Engagement Playbook

### First Hour After Posting
1. Reply to every comment (engagement boost)
2. Quote-tweet with additional context
3. Share in Discord/Telegram
4. Tag relevant ecosystem accounts

### Weekly Engagement Routine
- **Monday**: Share weekly goals / plans
- **Tuesday**: Technical deep-dive
- **Wednesday**: Community highlight / user story
- **Thursday**: Partner / ecosystem content
- **Friday**: Week recap / wins
- **Saturday**: Behind the scenes / team content
- **Sunday**: Rest or light engagement

### Hashtag Rotation
Rotate between these hashtag sets to avoid shadowbanning:
- Set A: #Solana #SolanaEcosystem #DeFi #Web3
- Set B: #SolanaBuilder #Crypto #Blockchain #Innovation
- Set C: #SolanaSummer #Web3Build #CryptoCommunity

## Key Principles

1. **Consistency Over Virality**: Post regularly (3-5x/week) rather than chasing viral moments
2. **Value Over Hype**: Educate and inform, don't just hype
3. **Community Over Broadcast**: Engage in conversations, don't just broadcast
4. **Solana-Native**: Reference the ecosystem, partner with projects, use ecosystem channels
5. **Authenticity Over Polish**: Real, genuine content beats overproduced corporate content
